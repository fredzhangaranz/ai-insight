## ‚úÖ Design Specifications

### Poster Dimensions

- **Size**: 2 meters (width) √ó 1 meter (height)
- **Ratio**: 2:1 (landscape orientation)
- **Resolution**: 300 DPI minimum
- **Color Mode**: CMYK (for printing)
- **Bleed**: 3cm on all sides

### File Requirements

- **Format**: PDF (print-ready)
- **Fonts**: Embedded or outlined
- **Images**: High-resolution (no pixelation)
- **QR Codes**: Minimum 8cm √ó 8cm (scannable from 1 meter)

### Readability

- **Title**: Readable from 3 meters
- **Section Headers**: Readable from 2 meters
- **Body Text**: Readable from 1.5 meters
- **Color Contrast**: Minimum 4.5:1 ratio (WCAG AA)

## üí° Pro Tips

1. **Readability Test**: Print a scaled-down version (A3) and view from 2 meters to test legibility
2. **QR Code Placement**: Bottom right, large (10cm √ó 10cm minimum), high contrast background
3. **Color Proofing**: Request a color proof from printer before final run‚Äîcolors look different in print vs. screen
4. **Backup Plan**: Have digital version on tablet in case of poster damage during travel
5. **Photo Opportunity**: Design includes space for CEO to stand beside without blocking content
6. **Lighting**: Bring portable LED light if booth lighting is poor
7. **Engagement Hook**: Consider "Ask us a question, get a [branded item]" to drive traffic
